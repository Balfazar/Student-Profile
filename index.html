Project Idea

Raspberry Pi Curtain controller with remote/Light senor

Overview

The Project idea that I thought of is an automated sunlight censored curtain opener. It consists of a UV light sensor which detects sunlight causing the curtain to automatically open without contact. This encourages better sleeping habits which is a vital element in everyone's lives. The project will consist of a raspberry pi which will be programmed using a programming language named python.

Motivation

Studies have shown that “waking up to sunlight directly places your body's inner clock ready for the day also known as your circadian rhythm.” Sunlight sends a mental signal for your body to wake up and start becoming alert. Whereas when the lights start to become dimmer during nighttime the body will naturally start to become sleepy again. The ongoing health benefits for waking up to sunlight supports the automated curtain opener. With the help of raspberry pi, the automated sunlight censored curtain opener can help build healthy sleeping habits for millions around the world.

Description

The automated project is to be created using raspberry pi and a programming language named python. The automated machine will be programmed to follow a set of commands. The commands include the opening of the curtain when the UV sensor is detected as the sunlight. The curtain will then automatically open itself reveling a bright room of sunshine. When the UV sensor no longer can detect a sunlight source then it will send a command to close the curtains back up. The UV sensors can be replaced or added on to a wireless remote controller, or a voice-controlled command using Alexa. The machine will be running on a micro servo motor and will be in a set rotating motion when a command is received. The gears will act accordingly to the commands whether it's pushing away or pulling in. A remote controller will also be attached to the machine as another way of functioning the curtains for other reasons (closing due to sunshine). The automated curtains will better impact your sleep quality and ensure that you wake up fully energized and focused for the day. Its role is to naturally fix your circadian rhythm. The moment the curtain closes or opens gives a mental reminder to the person that it is time to wake up or go to bed. Another feature is the motion sensor is used to determine the dimensions of any curtain holder and automatically stop or go for the commands to take place. A virtual app can also be created for setting up a timer or simply just normal control through Wi-Fi connection for more convenience.

Tools and Technologies

The hardware required for this project consist of having a Raspberry pi, a servo motor, raspberry Pi USB wireless adapter, UV light sensor, a microphone and a remote controller. The software required for programming the project will have Raspberry Pi 2 Noobs, Apache web server and wiring Pi.

Skills Required

Skills that will be needed to accomplish a successful automated curtain machine include having the beginner knowledge to code using python. Being familiar with electricity can be an advantage when dealing with automations such as the servo motors. Raspberry pi is a cheap and easily accessible tool that is used in this project that also requires basic knowledge.

Outcome

The finalized project will encourage healthy sleeping habits in an efficient and effective manner. Waking up to sunlight has shown to have many health benefits for the body and mind. Having an automated curtain controller will naturally fix your circadian rhythm by opening and closing your curtains by detecting sunlight. This can not only help people with sleeping disorders but will even benefit normal people.
